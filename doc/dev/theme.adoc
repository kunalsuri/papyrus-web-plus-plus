== Theme
The theme is defined in the file `./frontend/papyrus-web/src/theme/papyrusTheme.ts`. To create it, we started with the SysON theme one that uses less pink colors than the default theme provided by Sirius-Web &#128516;.
This file is referenced by  in the file `./frontend/papyrus-web/src/index.tsx.`

You will find some examples of usages below.

=== Primary Color Examples

*Definition*

image::theme/primary_definition.png[title="Primary Colors definition", align="left"]

*Usages*

These values are used on the **Create a new project** page.

* `primary.main`
** the border of the edited field
** active area
** button selected by default
** selected value in ComboBox, but lighter
* `primary.dark`
** hover for a button
* `primary.light`
** check box in property views
