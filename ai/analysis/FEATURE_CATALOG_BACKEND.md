## §2 Database Schema & Persistence  `[inferred]`

| Table | Mapped by | Created by | Stores |
|---|---|---|---|
| `profile` (id UUID PK, content TEXT) | ProfileResourceEntity — `backend/papyrus-web-domain/src/main/java/org/eclipse/papyrus/web/domain/boundedcontext/profile/ProfileResourceEntity.java` | Liquibase: `backend/papyrus-web-infra/src/main/resources/db/changelog/papyrus-web-schema.sql` via `backend/papyrus-web-infra/src/main/resources/db/changelog/papyrus-web.db.changelog.xml` | Published dynamic UML profiles as raw XML text |

That is the ONLY Papyrus-owned table. All other persistence (projects, semantic
documents, representations, libraries…) belongs to Sirius Web's own schema, created by
Sirius Web jars (external). Connection config: `backend/papyrus-web/src/main/resources/application.properties`
(`jdbc:postgresql://localhost:5439/papyrus-web-db`, Liquibase changelog pointer,
`sirius.web.enabled=*` with `studio` disabled).
